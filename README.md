# Wallpaper Hacker 
wallpaper for windows, with interactive hacker theme - using lively wallpaper

This project offers an animated hacker-themed wallpaper for Windows, fully compatible with **Lively Wallpaper**.
Ideal for those seeking a dynamic, modern, and hacker-inspired look on their desktop.

# How to use  
> 1. [Install Lively Wallpaper](https://apps.microsoft.com/detail/9ntm2qc6qws7) <br>
- <img src="images/install-lively.png" alt="install lively">
<br>
> 2. Download the files for this project. <br>
- <img src="images/download.png" alt="download project">
<br>
> 3. Open Lively → Select Matrix Rain Customizable → Open file location <br>
- 3.1 Delete all files from the directory. <br>
- 3.2 Extract the project files to the directory, and unpack `node_modules`. <br>
- <img src="images/open-directory.png" alt="open directory for edit">
<br>
> 4. Apply the wallpaper <br>
- 4.1 In the `Lively app`, click on Set as Wallpaper. <br>
- <img src="images/set-wallpaper.png" alt="set wallpaper project windows">
<br>
> 5. Enjoy your hacker-style live wallpaper! <br>
- <img src="images/live-wallpaper.png" alt="live wallpaper project">
